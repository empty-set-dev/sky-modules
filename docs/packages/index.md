# Packages

Sky Modules packages are published on NPM under the `@sky-modules` scope.

## Core Packages

### @sky-modules/core

Core utilities and helpers for TypeScript development.

**Features:**
- Array extensions (last, remove, shuffle)
- Type guards and utility types
- Event emitters and callbacks
- Async utilities and promise pools
- Environment and runtime detection

**Installation:**
```bash
npm install @sky-modules/core
```

**NPM:** [View on NPM](https://npmjs.com/package/@sky-modules/core)

## Framework Integrations

### @sky-modules/universal

Cross-framework components compiled with Mitosis for React, Vue, Solid, Svelte, Qwik, and Angular.

**Features:**
- Universal button components
- Form elements
- Layout components
- Typography system
- Popover and SlotRoot

**Installation:**
```bash

```

**NPM:** [View on NPM](https://npmjs.com/package/@sky-modules/universal)
