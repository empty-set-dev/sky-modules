# Пакеты

Пакеты Sky Modules публикуются на NPM под scope `@sky-modules`.

## Основные пакеты

### @sky-modules/core

Основные утилиты и хелперы для TypeScript разработки.

**Возможности:**
- Расширения Array (last, remove, shuffle)
- Type guards и утилитарные типы
- Event emitters и callbacks
- Async утилиты и promise pools
- Определение окружения и runtime

**Установка:**
```bash
npm install @sky-modules/core
```

**NPM:** [Смотреть на NPM](https://npmjs.com/package/@sky-modules/core)

---

## Дизайн система

- Box компоненты

**NPM:** [Смотреть на NPM](https://npmjs.com/package/@sky-modules/design)

## Интеграции фреймворков

### @sky-modules/universal

Кроссфреймворк компоненты скомпилированные с Mitosis для React, Vue, Solid, Svelte, Qwik и Angular.

**Возможности:**
- Универсальные компоненты кнопок
- Элементы форм
- Компоненты лейаута
- Типографская система
- Popover и SlotRoot

**Установка:**
```bash
npm install @sky-modules/universal
```

**NPM:** [Смотреть на NPM](https://npmjs.com/package/@sky-modules/universal)
