# Solid Адаптер

<div class="sky-gradient-text" style="font-size: 1.2em; margin: 1em 0;">
  Solid utility module
</div>

Платформенные компоненты и утилиты SolidJS для Sky Modules.

## Обзор

Solid-адаптер предоставляет SolidJS-специфичные реализации универсальных компонентов, скомпилированных через Mitosis. Используйте этот пакет для интеграции компонентов Sky Modules в Solid-приложения.

## Установка

```bash
npm install @sky-modules/solid solid-js
```

## Возможности

- **Универсальные компоненты** - Solid-реализации компонентов Sky Modules
- **Mitosis-компиляция** - Автоматическая генерация из определений универсальных компонентов
- **Типобезопасность** - Полная поддержка TypeScript
- **Реактивность** - Построен на детализированной реактивности SolidJS

## Использование

### Базовый пример

```tsx
import { Component } from '@sky-modules/solid'

function App() {
  return (
    <Component>
      Содержимое здесь
    </Component>
  )
}
```

## Компиляция компонентов

Компоненты в этом пакете скомпилированы из исходных файлов Mitosis (`.lite.tsx`), расположенных в директории `universal/`. Компиляция происходит автоматически во время сборки.

**Процесс сборки:**
```bash
# Компиляция Mitosis-компонентов в Solid
sky mitosis build <app-name>
```

## Пиринговые зависимости

- `solid-js` ^1.0.0

## Архитектура

Solid-адаптер является частью кросс-фреймворковой компонентной системы Sky Modules:
1. Универсальные компоненты определены в Mitosis
2. Скомпилированы в SolidJS во время сборки
3. Опубликованы как Solid-совместимый пакет

## Почему SolidJS?

SolidJS используется как основной фреймворк для:
- **Canvas JSX** - 2D canvas-рендеринг с реактивными графами сцен
- **Высокопроизводительный UI** - Детализированная реактивность без виртуального DOM
- **Малый размер бандла** - Компилируется в минимальный JavaScript
