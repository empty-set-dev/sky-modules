# sky run

Запуск Node.js скриптов с использованием среды выполнения Bun.

## Описание

Команда `run` позволяет выполнять Node.js скрипты напрямую с использованием среды выполнения Bun. Это полезно для запуска утилитных скриптов, миграций или любых отдельных JavaScript/TypeScript файлов.

## Использование

```bash
sky run <путь-к-скрипту>
```

## Аргументы

- `путь-к-скрипту` (обязательный) - Путь к файлу скрипта для выполнения

## Примеры

```bash
# Запустить скрипт из директории scripts
sky run scripts/migrate.ts

# Запустить утилитный скрипт
sky run utils/generate-data.js

# Запустить скрипт из конкретного пути
sky run ./tools/build-assets.ts
```

## Возможности

- **Среда выполнения Bun**: Быстрое выполнение с использованием Bun
- **Поддержка TypeScript**: Запуск TypeScript файлов напрямую без компиляции
- **Очистка консоли**: Автоматическая очистка консоли перед выполнением

## Примечания

- Консоль очищается перед выполнением скрипта
- Скрипты выполняются в контексте текущей рабочей директории
- Поддерживаются как JavaScript (.js), так и TypeScript (.ts) файлы
- Переменные окружения загружаются перед выполнением скрипта
