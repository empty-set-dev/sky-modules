# sky node

Управление Node.js приложениями с использованием среды выполнения Bun.

## Описание

Команда `node` предоставляет инструменты для разработки и запуска Node.js приложений с использованием среды выполнения Bun, обеспечивая быстрое выполнение и современную поддержку JavaScript/TypeScript.

## Использование

```bash
sky node <команда> [опции]
```

## Доступные подкоманды

### init

Инициализация нового Node.js приложения.

```bash
sky node init <имя-приложения>
```

**Аргументы:**
- `имя-приложения` (обязательный) - Имя Sky приложения для инициализации

**Пример:**
```bash
sky node init my-node-app
```

### dev

Запуск Node.js приложения в режиме разработки с автоперезагрузкой.

```bash
sky node dev <имя-приложения> [аргументы..]
```

**Аргументы:**
- `имя-приложения` (обязательный) - Имя Sky приложения для запуска
- `аргументы` (опциональные) - Дополнительные аргументы для передачи в приложение

**Пример:**
```bash
sky node dev my-app
sky node dev my-app --debug
sky node dev my-app arg1 arg2 arg3
```

### start

Запуск Node.js приложения в продакшен-режиме.

```bash
sky node start <имя-приложения> [аргументы..]
```

**Аргументы:**
- `имя-приложения` (обязательный) - Имя Sky приложения для запуска
- `аргументы` (опциональные) - Дополнительные аргументы для передачи в приложение

**Пример:**
```bash
sky node start my-app
sky node start my-app --port 8080
sky node start my-app config.json
```

## Возможности

- **Среда выполнения Bun**: Использует Bun для быстрого выполнения JavaScript/TypeScript
- **Автоперезагрузка**: Режим разработки включает автоматический перезапуск при изменении файлов
- **Поддержка TypeScript**: Нативная поддержка TypeScript без этапа компиляции
- **Передача аргументов**: Передача пользовательских аргументов в приложение

## Пример рабочего процесса

```bash
# Инициализировать новое Node.js приложение
sky node init my-backend-app

# Запустить в режиме разработки
sky node dev my-backend-app

# Запустить в продакшен-режиме
sky node start my-backend-app
```

## Примечания

- Консоль автоматически очищается при запуске команд dev или start
- Аргументы, переданные после имени приложения, пересылаются в ваше приложение
- Режим разработки включает отслеживание файлов для автоматического перезапуска
- Bun предоставляет нативную поддержку TypeScript и быстрое разрешение модулей
