# sky create

Create various types of modules and components for Sky Modules projects.

## Description

The `create` command provides templates for quickly scaffolding different types of modules, namespaces, and components in your Sky Modules project. It uses predefined boilerplates to ensure consistent project structure.

## Usage

```bash
sky create <type> <path> [options]
```

## Available Subcommands

### global-module

Create a global module with TypeScript definitions.

```bash
sky create global-module <module-path>
```

**Arguments:**
- `module-path` (required) - Path where the module will be created

**Example:**
```bash
sky create global-module src/modules/MyGlobalModule
```

### global-namespace

Create a global namespace for organizing related modules.

```bash
sky create global-namespace <namespace-path>
```

**Arguments:**
- `namespace-path` (required) - Path where the namespace will be created

**Example:**
```bash
sky create global-namespace src/namespaces/MyNamespace
```

### global-single-module

Create a single-file global module.

```bash
sky create global-single-module <module-path>
```

**Arguments:**
- `module-path` (required) - Path and name of the module file

**Example:**
```bash
sky create global-single-module src/modules/MySingleModule
```

### global-single-react-component

Create a single-file global React component with TypeScript definitions.

```bash
sky create global-single-react-component <react-component-path>
```

**Arguments:**
- `react-component-path` (required) - Path and name of the React component

**Example:**
```bash
sky create global-single-react-component src/components/MyComponent
```

### module

Create a standard module with index and implementation files.

```bash
sky create module <module-path>
```

**Arguments:**
- `module-path` (required) - Path where the module will be created

**Example:**
```bash
sky create module src/modules/MyModule
```

### namespace

Create a standard namespace structure.

```bash
sky create namespace <namespace-path>
```

**Arguments:**
- `namespace-path` (required) - Path where the namespace will be created

**Example:**
```bash
sky create namespace src/namespaces/MyNamespace
```

### react-component

Create a React component with TypeScript.

```bash
sky create react-component <react-component-path>
```

**Arguments:**
- `react-component-path` (required) - Path and name of the React component

**Example:**
```bash
sky create react-component src/components/MyComponent
```

## Default Behavior

If no subcommand is specified, the `create` command will initialize a complete workspace with all necessary configurations.

```bash
sky create
```

## Notes

- All paths are relative to the current working directory
- The command will not overwrite existing files (uses `force: false`)
- Module and component names are automatically extracted from the provided paths
- Templates include placeholder replacements for consistent naming
