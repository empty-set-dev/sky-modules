# sky

Основной интерфейс командной строки для фреймворка Sky Modules.

## Описание

`sky` - это основной инструмент CLI, который предоставляет доступ ко всем командам и функциональности Sky Modules. Он автоматически загружает конфигурации окружения и предоставляет полный набор подкоманд для управления проектом, разработки и развертывания.

## Использование

```bash
sky <команда> [опции]
```

## Доступные команды

- `create` - Создание различных типов модулей и компонентов
- `init` - Инициализация конфигураций проекта
- `node` - Управление Node.js (Bun) приложениями
- `web` - Управление веб-приложениями (Vike)
- `desktop` - Управление десктопными приложениями (Tauri)
- `ios` - Управление iOS приложениями (Expo)
- `android` - Управление Android приложениями (Expo)
- `add` - Добавление внешних модулей в проект
- `run` - Запуск Node скриптов с помощью Bun
- `test` - Запуск тестов с помощью Jest
- `format` - Форматирование кода с помощью eslint и stylelint
- `check` - Проверка кода
- `readme` - Конвертация MDX файлов в MD формат

## Опции

- `-h, --help` - Показать справочную информацию
- `-v, --version` - Показать версию

## Конфигурация окружения

CLI автоматически загружает переменные окружения из:
- `.env`
- `.env.[NODE_ENV]`
- `.env.local`
- `.env.[NODE_ENV].local`

## Примеры

```bash
# Создать новый модуль
sky create module my-module

# Инициализировать конфигурацию проекта
sky init

# Запустить сервер разработки для веб-приложения
sky web dev my-app

# Запустить тесты
sky test

# Отформатировать код
sky format
```
