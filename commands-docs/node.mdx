# sky node

Manage Node.js applications using Bun runtime.

## Description

The `node` command provides tools for developing and running Node.js applications using the Bun runtime, offering fast execution and modern JavaScript/TypeScript support.

## Usage

```bash
sky node <command> [options]
```

## Available Subcommands

### init

Initialize a new Node.js application.

```bash
sky node init <app-name>
```

**Arguments:**
- `app-name` (required) - Name of the Sky app to initialize

**Example:**
```bash
sky node init my-node-app
```

### dev

Run Node.js application in development mode with auto-reload.

```bash
sky node dev <app-name> [args..]
```

**Arguments:**
- `app-name` (required) - Name of the Sky app to run
- `args` (optional) - Additional arguments to pass to the application

**Example:**
```bash
sky node dev my-app
sky node dev my-app --debug
sky node dev my-app arg1 arg2 arg3
```

### start

Run Node.js application in production mode.

```bash
sky node start <app-name> [args..]
```

**Arguments:**
- `app-name` (required) - Name of the Sky app to start
- `args` (optional) - Additional arguments to pass to the application

**Example:**
```bash
sky node start my-app
sky node start my-app --port 8080
sky node start my-app config.json
```

## Features

- **Bun Runtime**: Uses Bun for fast JavaScript/TypeScript execution
- **Auto-reload**: Development mode includes automatic restart on file changes
- **TypeScript Support**: Native TypeScript support without compilation step
- **Argument Passing**: Pass custom arguments to your application

## Workflow Example

```bash
# Initialize a new Node.js app
sky node init my-backend-app

# Run in development mode
sky node dev my-backend-app

# Run in production mode
sky node start my-backend-app
```

## Notes

- Console is automatically cleared when starting dev or start commands
- Arguments passed after the app name are forwarded to your application
- Development mode includes file watching for automatic restarts
- Bun provides native TypeScript support and fast module resolution
