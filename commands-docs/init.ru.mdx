# sky init

Инициализация и конфигурация различных аспектов проекта Sky Modules.

## Описание

Команда `init` помогает настроить и сконфигурировать различные части вашего проекта Sky Modules, от базовых конфигурационных файлов до настроек среды разработки.

## Использование

```bash
sky init [команда] [опции]
```

## Доступные подкоманды

### sky-config

Инициализация файла конфигурации Sky.

```bash
sky init sky-config
```

Создает `.sky/sky.config.ts` с конфигурацией проекта по умолчанию.

### package.json

Обновление файла package.json проекта с зависимостями и скриптами Sky Modules.

```bash
sky init package.json
```

### ts-configs

Обновление файлов конфигурации TypeScript для всех модулей, примеров и приложений.

```bash
sky init ts-configs
```

Обеспечивает правильную настройку всех конфигураций TypeScript по всему проекту.

### packages

Инициализация пакетов и конфигураций.

```bash
sky init packages
```

Настраивает необходимые пакеты и их конфигурации.

### .gitignore

Инициализация или обновление файла .gitignore.

```bash
sky init .gitignore
```

Создает файл .gitignore, если он не существует, с соответствующими шаблонами для проектов Sky Modules.

### vscode-workspace

Обновление конфигурации рабочего пространства VSCode в корне проекта.

```bash
sky init vscode-workspace
```

Настраивает параметры рабочего пространства VSCode для оптимального опыта разработки.

### vscode-workspace-tasks

Обновление задач рабочего пространства VSCode для конкретного приложения.

```bash
sky init vscode-workspace-tasks <имя-приложения>
```

**Аргументы:**
- `имя-приложения` (обязательный) - Имя Sky приложения

**Пример:**
```bash
sky init vscode-workspace-tasks my-app
```

## Поведение по умолчанию

Запуск `sky init` без подкоманды инициализирует все конфигурации:

```bash
sky init
```

Это эквивалентно последовательному запуску всех подкоманд инициализации.

## Примеры

```bash
# Инициализировать всё
sky init

# Инициализировать только конфигурацию Sky
sky init sky-config

# Обновить package.json
sky init package.json

# Настроить конфигурации TypeScript
sky init ts-configs

# Инициализировать рабочее пространство VSCode для приложения
sky init vscode-workspace-tasks my-app
```

## Примечания

- Команды инициализации безопасно запускать несколько раз
- Существующие конфигурации будут обновлены, а не перезаписаны
- Поведение по умолчанию обеспечивает полную настройку проекта
