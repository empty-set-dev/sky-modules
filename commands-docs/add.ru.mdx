# sky add

Добавление внешних модулей в проект Sky Modules.

## Описание

Команда `add` позволяет интегрировать внешние модули в ваш проект Sky Modules. Это полезно для включения сторонних модулей или модулей из других проектов.

## Использование

```bash
sky add <путь-к-внешнему-модулю>
```

## Аргументы

- `путь-к-внешнему-модулю` (обязательный) - Путь к внешнему модулю для добавления

## Примеры

```bash
# Добавить модуль из локального пути
sky add ../shared-modules/auth

# Добавить модуль из конкретной директории
sky add /path/to/external/module

# Добавить модуль из текущей структуры директорий
sky add ./external-modules/payment
```

## Примечания

- Путь к модулю может быть относительным или абсолютным
- Команда интегрирует модуль в систему модулей вашего проекта
- Убедитесь, что внешний модуль соответствует соглашениям Sky Modules
