# sky create

Создание различных типов модулей и компонентов для проектов Sky Modules.

## Описание

Команда `create` предоставляет шаблоны для быстрого создания различных типов модулей, пространств имен и компонентов в вашем проекте Sky Modules. Она использует предопределенные шаблоны для обеспечения единообразной структуры проекта.

## Использование

```bash
sky create <тип> <путь> [опции]
```

## Доступные подкоманды

### global-module

Создание глобального модуля с TypeScript определениями.

```bash
sky create global-module <путь-к-модулю>
```

**Аргументы:**
- `путь-к-модулю` (обязательный) - Путь, где будет создан модуль

**Пример:**
```bash
sky create global-module src/modules/MyGlobalModule
```

### global-namespace

Создание глобального пространства имен для организации связанных модулей.

```bash
sky create global-namespace <путь-к-пространству-имен>
```

**Аргументы:**
- `путь-к-пространству-имен` (обязательный) - Путь, где будет создано пространство имен

**Пример:**
```bash
sky create global-namespace src/namespaces/MyNamespace
```

### global-single-module

Создание однофайлового глобального модуля.

```bash
sky create global-single-module <путь-к-модулю>
```

**Аргументы:**
- `путь-к-модулю` (обязательный) - Путь и имя файла модуля

**Пример:**
```bash
sky create global-single-module src/modules/MySingleModule
```

### global-single-react-component

Создание однофайлового глобального React компонента с TypeScript определениями.

```bash
sky create global-single-react-component <путь-к-react-компоненту>
```

**Аргументы:**
- `путь-к-react-компоненту` (обязательный) - Путь и имя React компонента

**Пример:**
```bash
sky create global-single-react-component src/components/MyComponent
```

### module

Создание стандартного модуля с индексным файлом и файлами реализации.

```bash
sky create module <путь-к-модулю>
```

**Аргументы:**
- `путь-к-модулю` (обязательный) - Путь, где будет создан модуль

**Пример:**
```bash
sky create module src/modules/MyModule
```

### namespace

Создание стандартной структуры пространства имен.

```bash
sky create namespace <путь-к-пространству-имен>
```

**Аргументы:**
- `путь-к-пространству-имен` (обязательный) - Путь, где будет создано пространство имен

**Пример:**
```bash
sky create namespace src/namespaces/MyNamespace
```

### react-component

Создание React компонента с TypeScript.

```bash
sky create react-component <путь-к-react-компоненту>
```

**Аргументы:**
- `путь-к-react-компоненту` (обязательный) - Путь и имя React компонента

**Пример:**
```bash
sky create react-component src/components/MyComponent
```

## Поведение по умолчанию

Если подкоманда не указана, команда `create` инициализирует полное рабочее пространство со всеми необходимыми конфигурациями.

```bash
sky create
```

## Примечания

- Все пути относительны к текущей рабочей директории
- Команда не перезаписывает существующие файлы (использует `force: false`)
- Имена модулей и компонентов автоматически извлекаются из указанных путей
- Шаблоны включают замену плейсхолдеров для единообразного именования
