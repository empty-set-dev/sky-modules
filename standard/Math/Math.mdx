## Math.minmax
```ts
const x = -9
Math.minmax(x, 0, 100) // 0
```

## Math.randomBetween(from: number, to: number): number
@param from: number - default 0<br/>
@param to: number - default 1<br/>
@returns — number
```ts
Math.randomBetween(0, 10) // 0 - 10
```

@param to — number<br/>
