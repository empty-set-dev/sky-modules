### Dev iOS (Expo)
```sh
sky ios dev [app-name]
```

### Dev android (Expo)
```sh
sky android dev [app-name]
```

### Build iOS (Expo)
```sh
sky ios build [app-name]
```

### Build Android (Expo)
```sh
sky android build [app-name]
```

### Start iOS (Expo)
```sh
sky ios start [app-name]
```

### Start android (Expo)
```sh
sky android start [app-name]
```
