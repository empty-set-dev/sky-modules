# @sky-modules/design

Компоненты дизайн-системы, токены и утилиты для стилизации на основе Panda CSS.

## Установка

```bash
npm install @sky-modules/design
```

## Возможности

- **Дизайн-токены** - Цвета, отступы, типографика, тени
- **Рецепты компонентов** - Переиспользуемые паттерны стилей
- **Система брендов** - Поддержка мультибрендовых тем
- **Компоненты макета** - Box, Layout, Grid
- **Panda CSS** - CSS-in-JS без рантайма с атомарным CSS
- **Типобезопасность** - Полная поддержка TypeScript

## Быстрый старт

### Использование дизайн-токенов

```typescript
import { css } from '@sky-modules/design/css'

const styles = css({
    backgroundColor: 'primary.500',
    padding: '4',
    borderRadius: 'md',
    color: 'white'
})
```

### Компонент Box

Универсальный компонент box с интеграцией дизайн-системы:

```tsx
import { Box } from '@sky-modules/design'

<Box
    backgroundColor="primary.500"
    padding="4"
    borderRadius="md"
    color="white"
>
    Содержимое здесь
</Box>
```

### Система брендов

Настройка нескольких брендовых тем:

```typescript
import { Brand } from '@sky-modules/design'

Brand.configure({
    default: {
        primary: '#007bff',
        secondary: '#6c757d'
    },
    darkMode: {
        primary: '#0056b3',
        secondary: '#5a6268'
    }
})
```

## Дизайн-токены

### Цвета
- Семантические цвета (primary, secondary, success, warning, error)
- Нейтральная палитра (gray.50 - gray.900)
- Брендовые цвета

### Отступы
- Консистентная шкала отступов (1 - 16)
- rem-единицы для доступности

### Типографика
- Семейства шрифтов, размеры, начертания
- Межстрочные интервалы и межбуквенные расстояния

## Компоненты

Пакет включает стилизованные компоненты, построенные на дизайн-системе:

- **Box** - Фундаментальный строительный блок со всеми CSS-свойствами
- **Layout** - Flex и grid макеты
- **Typography** - Заголовки, текст и параграфы

## Интеграция с Panda CSS

Дизайн-система построена на Panda CSS, что обеспечивает:

- **Нулевой рантайм** - Стили извлекаются во время сборки
- **Атомарный CSS** - Оптимизированный вывод CSS
- **Типобезопасность** - Автодополнение для всех дизайн-токенов
- **Рецепты** - Переиспользуемые варианты компонентов
