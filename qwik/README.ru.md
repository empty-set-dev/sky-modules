# Qwik Adapter

Qwik платформенные компоненты и утилиты для Sky Modules.

## Обзор

Qwik адаптер предоставляет Qwik-специфичные реализации универсальных компонентов, скомпилированных через Mitosis. Используйте этот пакет для интеграции компонентов Sky Modules в Qwik приложения.

## Установка

```bash
npm install @sky-modules/qwik @builder.io/qwik
```

## Возможности

- **Универсальные компоненты** - Qwik реализации компонентов Sky Modules
- **Mitosis-компиляция** - Автоматически генерируются из универсальных определений компонентов
- **Типобезопасность** - Полная поддержка TypeScript
- **Возобновляемость** - Построен на парадигме возобновляемости Qwik

## Использование

### Базовый пример

```tsx
import { component$ } from '@builder.io/qwik'
import { Component } from '@sky-modules/qwik'

export default component$(() => {
  return (
    <Component>
      Content goes here
    </Component>
  )
})
```

## Компиляция компонентов

Компоненты в этом пакете компилируются из Mitosis (`.lite.tsx`) исходных файлов, расположенных в директории `universal/`. Компиляция происходит автоматически во время сборки.

**Процесс сборки:**
```bash
# Компиляция Mitosis компонентов в Qwik
sky mitosis build <app-name>
```

## Одноранговые зависимости

- `@builder.io/qwik` ^1.0.0

## Архитектура

Qwik адаптер является частью кросс-фреймворк системы компонентов Sky Modules:
1. Универсальные компоненты определены в Mitosis
2. Компилируются в Qwik во время сборки
3. Публикуются как Qwik-совместимый пакет

## Почему Qwik?

Возобновляемость Qwik делает его идеальным для:
- **Мгновенная загрузка страниц** - Нулевые накладные расходы на гидратацию
- **Прогрессивная интерактивность** - Загрузка кода только при необходимости
- **Edge-first** - Оптимизирован для развертывания на edge
